# 🎛️ Media Compressor (Web)

[![CI](https://github.com/ShapArt/media-compressor-web/actions/workflows/ci.yml/badge.svg)](https://github.com/ShapArt/media-compressor-web/actions/workflows/ci.yml) [![license](https://img.shields.io/github/license/ShapArt/media-compressor-web)](https://github.com/ShapArt/media-compressor-web/blob/main/LICENSE)






**Ключевые факты:**


- 🎛️ Клиентский компресс фото через Canvas


- 🎞️ Видео через ffmpeg.wasm (подключаемый)


- 🔒 Нет бэкенда: приватность файлов у пользователя
